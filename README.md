# DropTime - 目薬管理アプリ v1.0

目薬の投与記録とスケジュール管理をするシングルページWebアプリケーション

## 📱 機能

- **今日タブ**: 毎日のチェックリストと投与記録（時刻付き）
- **履歴タブ**: 過去の投与記録と達成率
- **設定タブ**: 時間帯・目薬設定のカスタマイズ
- **5分タイマー**: 次の目薬までのカウントダウン
- **カラーカスタマイズ**: パッケージ色に合わせた視覚的識別

## 🎯 対象ユーザー

- 複数種類の目薬を定期的に使用する方
- iPhone/Apple Watchでのアラーム管理と併用
- シンプルで確実な記録管理を求める方

## 💻 技術仕様

- **純粋な静的サイト**: HTML5 + CSS + バニラJavaScript
- **ビルドツール不要**: そのままブラウザで動作
- **ローカルストレージ**: 端末内でのデータ永続化
- **モバイル最適化**: iPhone操作に特化したUI/UX

## 🚀 使い方

1. Webブラウザで `index.html` を開く
2. 設定タブで時間帯と目薬を設定
3. 今日タブで投与記録をつける
4. 履歴タブで過去の記録を確認

## 🎨 カスタマイズ

設定画面で以下をカスタマイズ可能：
- 朝・昼・夕・寝る前の時刻設定
- 目薬名・投与回数・パッケージカラー

## 📋 デフォルト設定

- **時間帯**: 07:00, 12:00, 18:00, 22:00
- **目薬A**: 1日4回（緑色）
- **目薬B**: 1日4回（青色）  
- **目薬C**: 1日2回（オレンジ色）
